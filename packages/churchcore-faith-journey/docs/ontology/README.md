# ChurchCore Faith Journey ontology docs

This package defines a **canonical individual faith journey** layer that derives from the ChurchCore upper ontology.

## What belongs here

- Canonical faith journey graphs (nodes/edges)
- Faith journey stage state categories

## T-Box / C-Box / A-Box layout

- **T-Box** (`ontology/tbox/`): schema (classes + properties)
- **C-Box** (`ontology/cbox/`): controlled vocabularies + canonical graphs
- **A-Box** (`ontology/abox/`): placeholder (real instance data lives in GraphDB named graphs)

## Quick navigation

- `journey.md`: canonical faith journey graph + stage states
