## What ChurchCore uses from this package

ChurchCore reuses AgenticTrust’s DnS-style primitives (all PROV-aligned):

- `at:Situation` / `at:SituationDescription`
- `at:Role`
- `at:SituationParticipation`

These are the base pattern ChurchCore specializes into `ccsit:*Situation` types and the spec/occ split.
