## What’s in this package (current)

The upper ontology provides the shared grammar aligned to:

- **PROV-O** (execution, occurrences)
- **p-plan / EP-PLAN** (specification and correspondence)
- **AgenticTrust** (DnS-style situations/roles/participation)

Key modules:

- `cc` (core: people/orgs/resources/activities + ActivityRole)
- `cc/situation` (situation family + spec/occ split)
- `cc/state` (time-varying concepts + manifestations + state categories)
- `cc/process` (ProcessRole vs Process)
- `cc/intent` (intent + inferred intent + strategic intent)
- `cc/community`, `cc/journey`, `cc/prov`, `cc/class`, `cc/commerce`
