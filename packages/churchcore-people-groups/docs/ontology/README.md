# ChurchCore People Groups ontology docs

This package defines **people group** concepts and a canonical **gospel progress phase** model (Phase 0–7).

## What belongs here

- People group entity modeling (ethnolinguistic groups, diaspora, nation/region/city scopes)
- Gospel progress phases as state categories
- A canonical phase journey graph

## T-Box / C-Box / A-Box layout

- **T-Box** (`ontology/tbox/`): schema (classes + properties)
- **C-Box** (`ontology/cbox/`): controlled vocabularies + canonical graphs
- **A-Box** (`ontology/abox/`): placeholder (real instance data lives in GraphDB named graphs)

## Quick navigation

- `people-groups.md`: people group model + phase states + graph usage
